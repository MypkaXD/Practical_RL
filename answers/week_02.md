# Что такое V(s), Q(s,a):

1. $v_{\pi}(s)$ - функция стоимости состояния (State Value Function) - мат ожидание суммарного дохода, который может получить агент, начиная с состояния $s$ и следуя своей политике $\pi$ (насколько крутое это состояние, и стоит ли вообще в нем находится при условии следования политике дальше)    
   2. $Q(s, a)$ - функция состояния и действия (Action-Value Function) - это ожидаемое вознаграждение, которое получит агент, если в состоянии s, агент совршит действие a, а дальше будет следовать своей политике.


$Q(s,a)=E[r+\gamma*V_{\pi}(s')|s,a]$
$V_{\pi}(s)=\sum_a{\pi(a|s)Q(s,a)}$

# Метод Policy Iteration (PI):

Метод PI - алгоритм решения MDP для поиска оптимальной стретегии.
1. Выбираем произвольную стратегию $\pi$ . $V(s)=0$ для всех $s$ 
2. Смотрим, насколько хорошая текущая стратегия $\pi$. Для этого итеративно обновляем ценность каждого состояния, пока разница между шагами, не станет меньше $\epsilon$ (невязка)
3. Улучшаем стратегию жадным образом $\pi(s)=argmax_aQ(s,a)$. Если стратегии совпадает со старой, то нашли оптимальную

Существует также VI (Value Iteration), она требует больше итераций, но каждая итерация завершается быстрее
