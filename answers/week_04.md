# DQN

Deep Q-Network - алгоритм RL, который объединяет Q-Learning и DeepLearning. Служит для работы с большими или непрерывными пространствами состояний.

Сеть на вход принимает состояние s и за один проход выдает предсказанное Q-value для любого действия из данного состояния s.

Loss function = $E[(r+\gamma * max_{a'}Q(s',a',\theta) - Q(s,a,\theta))^2]$ 

Для поиска точки минимума используется обычный GD (gradient descent): $W_{i+1}=W_{i}-{LearningRate}*\frac{dL}{dw}$ 

# Experience replay

Необходимо сохранить в ReplyBuffer и потом заиспользовать предыдущий опыт взаимодействия агента со средой $<s,a,r,s'>$, так как у DQN наблюдается проблема забывания старых взаимодействий
Также решает проблему кореллируемости данных для оптимизаторов NN с помощью добавления в историю случайной выборки из ReplyBuffer'a
Работает только с off-policy алгоритмами, для использования с on-policy нужно уменьшать размеры буффера, чтобы не накапливать слишком старые действия (с "плохой" политикой)

# Target Network

Вспомогательная нейронная сеть, которая позволяет решать проблему движущейся цели в DQN.

То есть создается копия сети с параметрами $\hat\theta$ , которые фиксируются на несколько итераций, потом обновляются. Целевое значение принимает следующий вид: $r+\gamma *max_{a}Q(s',a',\hat\theta)$ 

Есть два способа обновления весов $\hat\theta$ :
1) Hard update - раз в N шагов копируем веса из основной нейронной сети
2) Soft Update - на каждом шаге применяем регуляризатор для обновления весов $\hat\theta = (1-\alpha) * \hat\theta + \alpha*\theta$ 

# Double DQN:

Ввели для устранения ошибки переоценки в DQN. 

В DQN $target=r+\gamma*max_a′​Q(s′,a′;\hat\theta)$. Оператор max⁡ систематически переоценивает Qvalue, потому что:
1. Положительный шум в оценках $Q(s',a')$ усиливается через max
2. Отрицательный шум игнорируется
3. Ошибки накапливаются и распространяются

Вводят следующее:
1) Действие выбирается основной сетью $\hat{a}=argmax_a Q(s',a',\theta)$ 
2) Ценность действия оценивается другой сетью $Q(s', \hat{a}, \hat\theta)$ 
3) $target = r + \gamma*Q(s', \hat{a}, \hat\theta)$
